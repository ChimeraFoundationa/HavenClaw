## Reporting a Vulnerability

We take security vulnerabilities seriously. If you discover a security issue, please bring it to our attention right away!

### How to Report

**Please do NOT report security vulnerabilities through public GitHub issues.**

Instead, please report them via email at security@havenclaw.ai with the following information:

1. Description of the vulnerability
2. Steps to reproduce the issue
3. Potential impact
4. Suggested fix (if any)

### What to Expect

- **Initial Response**: You will receive an acknowledgment within 48 hours
- **Status Update**: We will provide a status update within 5 business days
- **Resolution**: We aim to resolve critical issues within 30 days

### Bug Bounty Program

We offer bug bounties for critical security vulnerabilities:

| Severity | Bounty Range |
|----------|--------------|
| Critical | $5,000 - $25,000 |
| High | $1,000 - $5,000 |
| Medium | $250 - $1,000 |
| Low | $50 - $250 |

### Security Best Practices

When using HavenClaw, please follow these security best practices:

1. **Never share your private key**
2. **Use a dedicated wallet** for agent operations
3. **Enable 2FA** on all connected channels
4. **Keep your software updated** to the latest version
5. **Verify contract addresses** before interacting
6. **Use environment variables** for sensitive data
7. **Review skill permissions** before installing
