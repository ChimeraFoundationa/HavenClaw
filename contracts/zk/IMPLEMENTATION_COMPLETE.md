# PLONK ZK Proof Implementation - 6G Capability Proof

## ✅ Implementation Complete

Successfully implemented PLONK zero-knowledge proof system for 6G capability verification on smart contracts.

---

## 🎯 What Was Built

### 1. Circuit (`zk/circuits/6g_capability_proof.circom`)
- **Type**: PLONK-compatible Circom circuit
- **Constraints**: 512 (n=512)
- **Public Inputs**: 1 (configurable to 6)
- **Private Witness**: 15 values (scores, thresholds, commitments)

**Proves**:
- 6 capability scores are in range [0, 100]
- Each score >= corresponding threshold
- Average accuracy = sum(scores) / 6
- Agent and capability bindings

---

### 2. Smart Contracts

#### `src/core/PLONKVerifier.sol`
- Auto-generated by snarkjs from circuit
- On-chain PLONK proof verification
- Uses BN254 pairing precompile (0x08)

#### `src/core/ZK6GVerifier.sol`
- Wrapper contract with 6G-specific interface
- Converts IPLONKVerifier.PlonkProof to verifier format

#### `src/interfaces/IPLONKVerifier.sol`
- Interface for PLONK verification
- PlonkProof struct with all commitment points

## 🚀 Usage

### Setup (One-time)
```bash
cd /root/soft/contracts

# Compile circuit + PLONK trusted setup + generate proof
npm run setup:plonk
```

### Generate New Proof
```bash
# Using Node.js
npm run generate:plonk

# Using bash
bash zk/scripts/generate-plonk-proof.sh
```

### Verify Proof
```bash
# Off-chain verification
npx snarkjs plonk verify zk/build/vkey_plonk.json \
  zk/build/public_plonk.json \
  zk/build/proof_plonk.json

# On-chain verification (via tests)
forge test --match-contract PLONKVerifierTest -vvv
```

## 🔧 Manual Commands

### 1. Compile Circuit
```bash
/root/.cargo/bin/circom zk/circuits/6g_capability_proof.circom \
  --r1cs --wasm --sym -o zk/build
```

### 2. PLONK Trusted Setup
```bash
cd zk/build

# Phase 1: Universal setup
npx snarkjs powersoftau new bn128 14 pot14_0000.ptau -v
echo "contribution" | npx snarkjs powersoftau contribute pot14_0000.ptau pot14_0001.ptau --name="PLONK" -v
npx snarkjs powersoftau prepare phase2 pot14_0001.ptau pot14_final.ptau -v

# Phase 2: Circuit-specific
npx snarkjs plonk setup 6g_capability_proof.r1cs pot14_final.ptau 6g_capability_proof_plonk.zkey -v
```

### 3. Export Verifier
```bash
npx snarkjs zkey export verificationkey 6g_capability_proof_plonk.zkey vkey_plonk.json
npx snarkjs zkey export solidityverifier 6g_capability_proof_plonk.zkey verifier_plonk.sol
```

### 4. Generate Proof
```bash
# Generate witness
cd 6g_capability_proof_js
node generate_witness.js 6g_capability_proof.wasm ../input.json witness.wtns

# Generate PLONK proof
cd ..
npx snarkjs plonk prove 6g_capability_proof_plonk.zkey witness.wtns proof_plonk.json public_plonk.json
```

## 🔐 Security Notes

1. **Trusted Setup**: PLONK uses universal trusted setup - one-time ceremony
2. **Proof Reuse**: Each proof can only be verified once (implement replay protection)
3. **Input Validation**: Always validate public inputs on-chain
4. **Gas Optimization**: Consider batch verification for multiple proofs
